# brown-bear-page
This project contains a basic HTML webpage that provides information about the brown bear (Ursus arctos). The page includes sections on species, habitat, physical features, and embedded media such as images and video. It is designed as a simple example of HTML structure, navigation, and content formatting.

Brown Bear Information Webpage

This repository contains a structured HTML webpage designed to present factual information about the brown bear (Ursus arctos). The page includes organized sections, internal navigation, and embedded media to provide a clear and informative overview of the species.

Features:

Well-structured and semantic HTML document
Navigation menu with links to section anchors
Dedicated sections for:
  Introduction
  Habitat
  Media
Inclusion of both an image and a video resource
Use of lists, headings, and descriptive text for clarity

Overview:

The brown bear is native to regions of North America and Eurasia and is classified as Least Concern according to conservation status assessments. This webpage summarizes:
    Several recognized subspecies
    Distinct physical characteristics of the species
    Countries with both large and small brown bear populations

Media:

The webpage incorporates:
    A representative image of a brown bear
    A short educational video providing additional context

Purpose:

This project serves as an example of a basic HTML webpage layout and may be used for:
    Practicing core HTML development skills
    Demonstrating webpage structure and internal linking
    Showcasing introductory web design techniques
